## Naming convention

Il nome del foglio deve essere nel formato:
"Scheda segnalazione Numero Nucleo Familiare (Nome Cognome Intestatario) <- Nome Inviante"
ad esempio:

* "Scheda segnalazione 5 (Mario Rossi) <- Lorenzo Mazza"

## Compilazione

La scheda di segnalazione va compilata dall'operatore che riceve il beneficiario.

## Organizzazione nel Drive

La cartella delle schede di segnalazione si trova in:

* Il mio Drive/La Bussola dell'Abitare/Beneficiari/Cartella Beneficiario/
* Il mio Drive/La Bussola dell'Abitare/Schede di segnalazione/

### Campi

| Campo | Istruzioni | Sorgente |
| --- | --- | --- |
| Inviante | Nome e Cognome Operatore che ha inviato la segnalazione |  |
| Nome | Nome del beneficiario |  |
| Cognome | Cognome del beneficiario |  |
| Data nascita | Data di nascita del beneficiario |  |
| Luogo nascita | Luogo di nascita del beneficiario |  |
| Codice Fiscale | Codice Fiscale del beneficiario |  |
| Cellulare | Cellulare del beneficiario |  |
| Bisogno | Bisogno del beneficiario |  |
| Descrizione | Descrizione del caso |  |
| Luogo e data | Luogo e data |  |
| Firma operatore | Firma dell'operatore |  |
